## 🚀 𝑴𝒂𝒊𝒏 𝑭𝒆𝒂𝒕𝒖𝒓𝒆𝒔
*   **Baileys Core:** يعتمد على أحدث إصدارات Baileys لضمان استقرار الاتصال.
*   **Event Handling:** نظام متكامل للتعامل مع الرسائل، الإضافات، والطرد بمرونة عالية.
*   **Interactive System:** يدعم الـ Interactive Messages والـ Buttons بشكل احترافي.
*   **Smart Mentions:** نظام ذكي لمعالجة الـ `@lid` وحل مشاكل المنشن في الجروبات الكبيرة.
*   **Aesthetic UI:** كروت ترحيب (Welcome/Bye Cards) مصممة بتقنية Canvas مع دعم النيون.
*   **Group Security:** نظام طرد سريع (Kick) وحماية للجروبات من التخريب.
